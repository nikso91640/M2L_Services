# Captures d'écran — M2L Services

Ce dossier contient les captures des écrans de l'application pour la documentation E6.

## Script fourni

- `capture-ecrans.ps1` : script PowerShell qui capture l'écran principal en PNG.

## Utilisation

1. Ouvrir PowerShell dans ce dossier.
2. Exécuter :

```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
.\capture-ecrans.ps1
```

3. Suivre les instructions du script :
   - afficher l'écran demandé dans l'application,
   - appuyer sur Entrée pour capturer.
